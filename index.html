🔍 SENSORES FI (MAP, O2, IAC)

Falla Sensor MAP: Moto se ahoga al acelerar fuerte.
Falla Sensor O2: Humo negro y consumo excesivo.
Falla Válvula IAC: Se apaga sola en semáforos (mínima).
TIP: Si falla el ralentí, limpia la IAC con Contact Cleaner, ¡nunca con gasolina!

🧰 HERRAMIENTAS PRO

OBD2 + 6 Pines: Conecta bajo el asiento para borrar "Check Engine".
Torquímetro: Vital para culata (25Nm) y ejes (90Nm).
Copa Torre: Para soltar la tuerca del filtro centrífugo y embrague.

🛢️ FLUIDOS E INGENIERÍA

Aceite Sugerido: 10W50 o 20W50 (Sintético) JASO MA2.
Horquillas (Barras): Cambiar aceite cada 15,000 km (Aceite 10W o 15W).
Embrague: Si patina al acelerar a fondo, toca cambio de discos.

⚡ SISTEMA ELÉCTRICO

Caja Fusibles: Bajo el asiento del conductor. Revisa si no prende tablero.
ECU: No desconectar sensores con el switch abierto o quemarás la compu.

⚠️ PROTOCOLO ELÉCTRICO SEGURO

1. PARA RETIRAR (Desconectar): Primero el NEGRO (-), luego el ROJO (+).

2. PARA INSTALAR (Conectar): Primero el ROJO (+), luego el NEGRO (-).

Razón: Evita cortocircuitos si la herramienta toca el chasis mientras manipulas el polo positivo.
